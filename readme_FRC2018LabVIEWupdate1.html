NI Software for FRC 2018 Update Readme

November 2017

This file contains important information about the NI Software for FRC 2018 Update.

Overview

The NI Software for FRC 2018 Update is an update to the NI Software for FRC 2018 included in the FRC kit of parts. This update installs the following:

  • (LabVIEW users only) LabVIEW Update
  • FRC Driver Station
  • FRC Utilities

System Requirements

If you choose to program in LabVIEW, you must install the NI Software for FRC 2018 included in the FRC kit of parts (or online) before you can install this update. The NI Software for FRC 2018 Update has the following minimum requirements:

  • Windows 10/8.1/7 SP1

After installation you must run NI License Manager to activate the Vision Development Module Runtime Engine using the Serial Number provided in the FRC kit of parts. If you installed and activated the NI Software for FRC 2018 prior to this update, then the Vision Development Module Runtime Engine license will already be activated.

Fixed Issues

Refer to the National Instruments FRC community website at ni.com/frc for information about known issues, bug fixes, and troubleshooting LabVIEW for FRC. Refer to the FIRST website at www.firstinspires.org/robotics/frc for information about known issues in components of the NI Software for FRC 2018.

Installing the Update

Follow the steps on the ni.com/frc website to install the NI Software for FRC 2018 Update.

Refer to the FIRST website at www.firstinspires.org/robotics/frc for more information about the NI Software for FRC 2018 and related updates.
